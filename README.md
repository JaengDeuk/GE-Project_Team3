# Control
이동 : w, a, s, d <br>
대시 : shift <br>
색상변경 : 1, 4 <br>
아이템박스 상호작용 : e <br>
인벤토리 : i <br>
점프 : spacebar (어빌리티 장착 후 사용가능) <br>
점멸 : alt (어빌리티 장착 후 사용가능) <br>
공격 : mouse L (장비 장착 후 사용가능) <br>
막기 : mouse R (장비 장착 후 사용가능) <br>
